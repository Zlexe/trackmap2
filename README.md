# TrackMap — Руководство по запуску и работе

## 🚀 Быстрый старт

1. **Установите VS Code**: https://code.visualstudio.com/download
   Или через Homebrew: `brew install --cask visual-studio-code`

1. **Клонируйте проект** в папку на Desktop:
   ``bash
   cd ~/Desktop
   git clone <REPO_URL> trackmap-2-копия
   ``

1. **Откройте терминал внутри VS Code**:
- Перейдите в папку проекта
- Откройте VS Code (**Cmd+Shift+E → Open Folder**)
- Выберите папку проекта
- Откройте терминал (**View → Terminal** или `Cmd+`)

1. **Установите зависимости**:
   ``bash
   npm install
   ``

1. **Запустите разработку**:
   ``bash
   npm run dev
   ``

1. **Откройте браузер**: http://localhost:5173

## 📋 Подробные инструкции

### Часть 1: Подготовка системы

#### Шаг 1: Node.js и npm

**Проверка:**
``bash
node --version
npm --version
``

**Установка через Homebrew:**
``bash
brew install node
``

**Установка из официального安装包:**
1. Перейдите на https://nodejs.org/
2. Скачайте LTS версию для macOS
3. Установите

#### Шаг 2: Установка зависимостей проекта

Откройте терминал, перейдите в папку проекта и выполните:
``bash
npm install
``

Это создаст папку `node_modules/` с нужными пакетами.
---

### Часть 2: Открытие в VS Code

**Метод А: Открытие как код (рекомендуется)**

Через терминал из любой директории:
\`\`bash
code "/Users/vadim/Desktop/trackmap 2 — копия"
\`\`

Или через macOS Spotlight (**Cmd+Space → Visual Studio Code**):
\`\`bash
open -a "Visual Studio Code" "/Users/vadim/Desktop/trackmap 2 — копия"
\`\`

**Метод Б: Через Finder (если метод А не работает)**

1. Откройте **Finder**
2. Перейдите в папку проекта
3. Нажмите \`Cmd+I\` (**Properties**) для любого файла проекта
4. В поле **"Open with"** выберите **Visual Studio Code**
5. Нажмите **"Set for all files"**
6. Закройте Finder

---

### Часть 3: Первый запуск разработки

#### Способ 1: Запуск из-под VS Code (рекомендуется)

1. Откройте папку проекта в VS Code (**File → Open Folder**)
2. Откройте терминал внутри VS Code (**View → Terminal** или \`Cmd+\`)
3. В терминале выполните:
   \`\`bash
   npm run dev
   \`\`
1. Внизу панели появится **"Local:5173"** — это порт разработки
2. Откройте http://localhost:5173 в браузере

#### Способ 2: Запуск из внешнего терминала (Mac Terminal/iTerm2)

Откройте любой терминал, выполните по очереди:

\`\`bash
cd "/Users/vadim/Desktop/trackmap 2 — копия"
npm install
npm run dev
\`\`

Появится сообщение **Local:5173**. Откройте http://localhost:5173.

#### Способ 3: Direct Node/Vite

Если \`npm run dev\` не работает, можно запустить напрямую через Node:

\`\`bash
node node_modules/vite/bin/vite.js
\`\`

---

### Часть 4: Частые проблемы и решения

**Проблема: "command not found: vite"**
- **Решение:** Выполните \`npm install\` в папке проекта.

**Проблема: "port 5173 is in use"**
- VS Code обычно автоматически предлагает другой порт.
- Можно указать порт вручную через переменную окружения:
  \`\`bash
  PORT=3000 npm run dev
  \`\`

**Проблема: "npm install failed with ERESOLVE"**
- Попробуйте с флагом \`--legacy-peer-deps\`:
  \`\`bash
  npm install --legacy-peer-deps
  \`\`
